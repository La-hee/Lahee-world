## 📅 오늘의 일기
요일별 짧은 일기와 함께 맛집 정보를 공유하고 별점을 기록한 내용을 담은 일기 웹

## 🚀 주요 기능 및 페이지 구성
### 1. 메인 페이지 (Main Page)
<img width="1711" height="648" alt="Image" src="https://github.com/user-attachments/assets/1ddbf1a8-dcb5-4626-8d05-6d642af25702" />

- 요일 내비게이션: 화면 중앙의 한자로 표기된 月, 火, 水, 木, 金, 土, 日 버튼이 배치되어 있음
- 각 요일을 클릭하면 해당 요일의 상세 일기 페이지로 이동함

### 2. 요일별 일기 페이지 
두 가지 타입의 레이아웃이 존재

- 타입 1 (기본형): 그날의 대표 사진 한 장과 짧은 일기 내용으로 구성되어 있음

- 타입 2 (음식점 공유형): 기본 일기 구성에 더해, 공유하고 싶은 맛집이 있는 경우 우측 하단에 음식점 상세 페이지로 이동하는 링크가 추가되어 있음

- 이동 기능: 상단의 '오늘은 ~요일' 문구를 클릭하면 다시 메인 페이지로 돌아갑니다.

### 3. 음식 상세 페이지
추천하고 싶은 음식점의 상세 정보를 담고 있음

- 콘텐츠: 음식 사진, 한 줄 평, 별점 기록.

- 네이버 지도 연동: '가는 방법' 섹션에 네이버 지도 API를 불러와 실제 위치를 확인할 수 있음.

- 인터랙티브 별점: 별점은 평소에 숨겨져 있으며, 버튼을 클릭해야만 별점이 나타나도록 구현해 재미 요소를 더함.

- 이동 기능: '맛있게 먹은 것' 문구를 클릭하면 메인 페이지로 이동함.

### 🛠 Tech Stack
Frontend: HTML, CSS


### 📂 Project Structure
- MainPage: 한자 요일 버튼 및 메인 레이아웃

- DayPage: 요일별 사진 및 일기 컴포넌트

- FoodPage: 지도 및 별점 토글 기능이 포함된 상세 페이지
